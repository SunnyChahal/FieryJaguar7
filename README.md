# CPEN311_CSSC
Repository for Corwin and Sunny CPEN311 labs

Using Git:

if you've added a new file or deleted an old one,
        $ git add .
        $ git commit -m "message"
        $ git push
otherwise,
        $ git commit -a -m "message"
        $ git push
to bring in changes from online repository,
        $ git pull
to check your changes,
        $ git status
